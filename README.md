# Sword Art Online: Hollow Fragment Internal Hack

## Current features:

- Infinite health
- Damage
- Find closest monster to player
- Teleport all enemies on map next to you in one clumped up ball
- Speed hack

## How to use:

- Download a DLL injector
- Download project and compile into Internal_Hack.dll
- Inject into Steam version of Sword Art Online: Hollow Fragment
- Use numpad to activate/deactivate hacks
